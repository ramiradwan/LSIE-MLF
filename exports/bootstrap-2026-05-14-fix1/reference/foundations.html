Foundations

Tokens for the dark, data-dense Operator Console shell. Three near-black neutrals form a tight surface ladder; one accent and five status hues carry every trust signal in the product. There are no gradients, no shadows, no decorative chrome.

Surface ladder

Hierarchy is built from three near-black greys plus a single border. The window background is the darkest layer; surfaces sit on top of it; raised surfaces sit on top of those. Borders separate panels — never shadows.

Status semantics — the five trust hues

The console's most important visual contract: recovering is not the same as broken, and degraded is not the same as error. Each hue is a distinct operator-facing state, never used decoratively.

Severity tints — banner backgrounds

Banners are the only place we tint a surface with the accent or a status hue. Tints are alpha-blended over the page background — never solid.

Type

A single sans family — Segoe UI on Windows, Inter as fallback — handles every label, value, and copy in the shell. Cascadia Mono handles the one place numbers must align: the action-bar countdown. Type sizes step in 1–2 px increments because the operator reads at desk distance for hours.

Radii

Two radii only. Small for inputs, buttons, banners, tables. Medium for panels and metric cards. No pill-shaped containers except the status pill itself.

Width bands

Responsive behavior is shared across pages: width bands at 640 and 980 reflow card grids and hide low-priority table columns. The ActionBar switches to compact below 1024. Pages may add local breakpoints (Live Session: 720/1040, Experiments: 760/1040, Health: 760/1120). The shell never reorganizes navigation — only density.
